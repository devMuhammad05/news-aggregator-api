# News Aggregator API

[![Tests](https://github.com/devMuhammad05/news-aggregator-api/actions/workflows/tests.yml/badge.svg)](https://github.com/devMuhammad05/news-aggregator-api/actions/workflows/tests.yml)

A Laravel-based application that fetches, processes, and serves articles from multiple news sources efficiently through a RESTful API. It uses jobs, DTOs, and lazy processing to handle large volumes of news data while maintaining reliability and performance.
